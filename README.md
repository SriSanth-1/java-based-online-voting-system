# Java Voting System

## Developed By
Srisanth – Java Developer

## Project Overview

A GUI-based online voting system developed using Java Swing.

This application allows users to vote for candidates and display voting results through an interactive graphical user interface.

---

## Features

- GUI-based voting system
- Vote counting
- Result display
- Interactive interface
- Java Swing application

---

## Technologies Used

- Java
- Swing
- OOP Concepts
- Event Handling
